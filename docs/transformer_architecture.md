# Hybrid transformer architecture

GraphCodeBERT is the default configurable `AutoModel`, used over structured architecture text—not source-code bodies. Its mask-mean semantic vector is fused with an MLP over normalized Arcan/Stage 1 features and embeddings for smell, component/construct type, and recommendation family. Shared fusion layers feed applicability, five-class or ordinal relevance, bounded benefit/risk, uncertainty, and embedding outputs.

Numeric-only, semantic-only, semantic+numeric, semantic+numeric+categorical, and full-baseline-score ablations are registered. Missing labels are masked task-by-task. This architecture cannot recover cohesion, behavior, deployment, or transactional semantics absent from Arcan data.
